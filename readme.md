# Node.js 백엔드 개념 교재

## 구성 원칙
- 실습/체크리스트/퀴즈를 제외하고 개념 설명에 집중
- 각 챕터를 독립적으로 읽어도 이해 가능한 구조
- 프론트 관점에서 백엔드 사고로 전환하는 설명 방식 유지
