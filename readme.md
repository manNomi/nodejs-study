# Node.js 섹션별 개념 교재

이 저장소는 프론트엔드 개발자가 Node.js 백엔드로 확장할 때 필요한 내용을 **섹션별로 분리된 설명형 문서**로 정리한 학습 노트입니다.

## 작성 원칙
- 각 섹션의 성격(입문/JS/코어/HTTP/Express/DB/배포/TS)에 맞춰 설명을 분리
- 실습 체크리스트보다 개념 설명과 맥락 이해에 집중
- 프론트 관점에서 백엔드 사고로 전환할 때 필요한 연결 포인트 포함
