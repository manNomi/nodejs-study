# Node.js 백엔드 교과서 노트 (프론트 개발자 전용)

목표: 프론트엔드 개발자가 Node.js 백엔드를 **직접 설계/구현/운영**할 수 있는 수준으로 성장하는 것.

## 이 노트의 사용법
1. 각 강의 `readme.md`에서 "이 장에서 답해야 할 질문" 3개를 먼저 읽습니다.
2. "교과서 핵심 개념"을 읽고, 요청-처리-응답 흐름을 다이어그램으로 그립니다.
3. "코드 워크스루"를 실행한 뒤 실패 케이스를 의도적으로 만들어 봅니다.
4. "실습 미션" 완료 기준 3개를 체크합니다.
5. "복습 문제"에 글로 답하며, 팀 규칙 문서로 확장합니다.

## 학습 원칙
- 기능 중심이 아니라 **장애 대응 중심**으로 학습
- 프론트/백 경계를 나눠서 책임 분리
- 코드 작성과 동시에 테스트/로그/배포를 함께 설계
